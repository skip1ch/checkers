##Дубовая Доска
Современная платформа для игры в русские шашки — с мультиплеером по ссылке, ИИ на minimax и разбором партий от AI.

##Для кого и зачем
Для тех, кому надоели существующие шашечные сайты — перегруженные рекламой, с устаревшим дизайном и без разбора. Здесь можно сыграть с другом в один клик по ссылке, потренироваться против ИИ трёх уровней — и после партии посмотреть, где ошибся.

##Фичи
**Игра**

Полные правила русских шашек: обязательное взятие, цепные захваты, дамки
Три режима: против ИИ, с другом онлайн, локально на одном экране
Шахматные часы — по 5 минут на каждого игрока, отсчёт идёт даже когда вкладка закрыта
Эмодзи-реакции во время партии (и ИИ иногда тоже реагирует — по ситуации)

**Искусственный интеллект**
ИИ написан с нуля — minimax с альфа-бета отсечением. Три уровня отличаются глубиной поиска и эвристикой оценки позиции:

Лёгкий — считает только материал
Средний — добавляет контроль центра, безопасность шашек, угрозы взятия
Сложный — глубокий поиск, приоритет дамок, давление на позицию

**Разбор партии**
После каждого матча — автоматический анализ: пропущенные взятия, критические ошибки, текстовые комментарии к ходам. График захватов по ходам (Chart.js). Любую прошлую партию можно пересмотреть в режиме видео повтора.

**История матчей**
Все сыгранные партии сохраняются в профиле: соперник, результат, изменение кубков, дата. Из истории можно сразу открыть разбор любого матча — даже того, что был сыгран неделю назад.

**Мультиплеер**
Supabase Realtime (broadcast по WebSocket). Комната создаётся за секунду, друг заходит по ссылке. Ходы, эмодзи, сдача, синхронизация доски — отдельные события. Если игрок вышел и вернулся, клиент отправляет state_request, соперник отвечает текущим состоянием.

**Профиль и прогрессия**
Кубки (рейтинг): +10 за победу, штраф за поражение
Гемы — внутренняя валюта, тратятся в магазине на темы и эмодзи
История матчей с результатами и быстрым переходом в разбор
Аватарка: цветной кружок с буквой по умолчанию, можно загрузить своё фото

**Дизайн**
8 визуальных тем через CSS custom properties — смена темы это буквально document.documentElement.style.setProperty(...), без пересоздания компонентов. Темы покупаются в магазине.

##Технически
Фронтенд	React 18, Vite
База данных	Supabase (PostgreSQL + Realtime + Auth)
Платежи	Stripe
Хостинг	Netlify
ИИ	Minimax + alpha-beta pruning, написан с нуля
Синхронизация	WebSocket broadcast, localStorage + Supabase как резервная копия

##Почему это ценно
Существующие шашечные сайты либо перегружены рекламой, либо сделаны 15 лет назад. Здесь — чистый продукт с современным UX, честным рейтингом, реальным онлайн-режимом и ИИ-разбором партий. Это не учебный проект — это прототип сервиса, который можно монетизировать (магазин уже работает через Stripe).
