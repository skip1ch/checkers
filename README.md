# Шашки

Веб-приложение для игры в русские шашки. Можно играть против ИИ, друга или случайного соперника онлайн.

**[Открыть проект](https://skip1ch.github.io/checkers)**

---

## Что умеет

**Игра**
- Полные правила русских шашек — обязательное взятие, дамки, многоходовые серии
- ИИ на трёх уровнях сложности (алгоритм minimax с alpha-beta отсечением)
- Матчмейкинг — автоматический поиск соперника с похожим рейтингом
- Игра с другом — приглашение по нику

**Прогрессия**
- Кубки — растут за победы, падают за поражения
- Пять арен — от Новобранца до Легенды, каждая открывается при определённом количестве кубков
- Серия дней — огонёк который растёт чем дольше заходишь каждый день

**Социальное**
- Друзья — список, статус онлайн, вызов на дуэль
- Кланы — вступление, клановый чат, клановый рейтинг
- Глобальный лидерборд — топ игроков по кубкам

**Дополнительно**
- Эмодзи во время игры — анимированно появляются на экране со звуком
- AI Coach — после каждой партии разбирает ошибки и даёт советы
- Магазин — паки эмодзи, скины фигур, темы досок, защита серии, PRO подписка
- Раздел обучения — пошаговые уроки по основам, тактике и окончаниям

---

## Технологии

- Vanilla HTML / CSS / JavaScript
- React 18 (через CDN, без сборщика)
- Игровой движок написан с нуля — без сторонних шашечных библиотек
- Web Audio API — звуки ходов и реакций

---

## Запуск

Просто открой `index.html` в браузере. Никаких зависимостей и сервера не нужно.
