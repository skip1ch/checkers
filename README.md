## О проекте

Checkers — это онлайн-платформа для игры в шашки, где пользователи могут создавать аккаунты, играть друг с другом в реальном времени и получать доступ к premium-функциям через Stripe-платежи.

Проект создан для школьников, студентов и любителей настольных игр, которым нужен простой, современный и удобный способ играть в шашки онлайн с друзьями или другими пользователями.

Основные функции платформы:
- онлайн-игра в реальном времени;
- регистрация и вход через Google;
- система premium-подписки;
- магазин тем и скинов для персонализации интерфейса;
- использование эмодзи и игровых реакций во время матчей;
- современный адаптивный интерфейс;
- система аккаунтов и хранения прогресса.

Главная ценность продукта — создание удобного, интерактивного и персонализированного игрового опыта, который объединяет соревновательный геймплей, социальные элементы и современные веб-технологии в одном приложении.
