# Описание

Русские шашки с мультиплеером по ссылке, ИИ на minimax и разбором партий.

# Для кого и зачем

Для тех, кому надоели существующие сайты с шашками — перегруженные рекламой,
с устаревшим дизайном и без нормального онлайн-режима. Здесь можно сыграть
с другом по ссылке без регистрации, или потренироваться против ИИ трёх уровней,
а после — посмотреть разбор своей партии.

# Что внутри технически

**ИИ** написан с нуля — minimax с альфа-бета отсечением. Три уровня отличаются
глубиной поиска и эвристикой оценки позиции: от подсчёта шашек до контроля
центра и угроз дамки.

**Мультиплеер** работает через Supabase Realtime (broadcast по WebSocket).
Ходы, эмодзи-реакции и сдача — отдельные события в комнате. Если игрок
вышел и вернулся, клиент отправляет `state_request`, соперник отвечает
текущим состоянием доски. Резервная копия — в Supabase-профиле.

**Разбор партии** — кастомный движок анализа проходит по истории ходов,
находит пропущенные взятия и ошибки, строит график захватов (Chart.js)
и даёт текстовые комментарии. Повтор любой прошлой партии — из истории матчей.

**Темы** — 8 штук через CSS custom properties: смена темы = переписать
переменные на `:root`, без пересоздания компонентов.


# Стек

React 18, Vite, Supabase (PostgreSQL + Realtime + Auth), Stripe, Netlify
